Standard PPS Calculation Components for 100Gbps Ethernet:

Link rate: 100 Gbps = 100,000,000,000 bits per second

Ethernet frame size: Variable, depending on payload/data size (including Ethernet header and FCS)
Physical layer overhead: 20 bytes total (preamble 7 bytes + SFD 1 byte + Inter-frame gap 12 bytes)

Total frame size on wire: Ethernet frame size + 20 bytes physical overhead
Bits per packet: Total frame size Ã— 8 bits/byte
PPS: Line rate (bps) / Bits per packet

Explanation:

The Ethernet frame sizes here range from minimum ethernet frame (64 bytes) to jumbo frames (up to ~9000 bytes).
Physical layer overhead (20 bytes) is added to account for preamble, SFD, and IFG.
PPS decreases as frame size increases because each packet carries more bits, hence fewer packets are needed to saturate the 100 Gbps line.
The calculation method follows IEEE 802.3 standards and is commonly used for realistic PPS measurement on 100GbE networks.
